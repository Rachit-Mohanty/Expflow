EXPFLOW 💸✨

Master Your Finances with EXPFLOW.
Take control of your spending through immersive 3D visualization, advanced analytics, and strategic budgeting. No sign-ups required.

🌟 Features

EXPFLOW isn't just another expense tracker; it's a highly visual, immersive financial dashboard built directly in the browser.

🏎️ 3D Hyperspeed Environment: A real-time, hardware-accelerated WebGL background powered by Three.js with custom post-processing bloom and distortion effects.

🍱 Smart Bento Analytics: Beautifully animated pie charts and dynamically scaling bar graphs that track your daily, monthly, and yearly cash flow using Chart.js.

🔒 Total Data Privacy: Runs 100% locally in your browser using HTML5 LocalStorage. No accounts, no external databases.

🌍 Global Currencies: Seamlessly switch between USD, EUR, GBP, INR, JPY, and more with a single click.

📥 Secure Backups (JSON): Export your transaction history to a raw JSON file and restore it on any device.

📄 PDF Invoices: Generate professional, theme-matched PDF financial statements of your transaction history instantly.

📱 Fully Responsive: A sleek, glassmorphic UI styled with Tailwind CSS that looks perfect on desktop, tablet, and mobile.

🛠️ Tech Stack

EXPFLOW is built using modern, lightweight web technologies without the need for a complex build step:

Frontend: HTML5, Vanilla JavaScript (ES Modules)

Styling: Tailwind CSS (via CDN)

3D Graphics: Three.js (v0.143.0) & postprocessing

Charts: Chart.js

PDF Generation: jsPDF & jsPDF-AutoTable

Icons & Typography: FontAwesome, Google Fonts (Saira Stencil One, Inter)

🚀 Getting Started

Since EXPFLOW is completely client-side and relies on ES Modules, you just need a basic local web server to run it (to avoid CORS errors with the module imports).

Prerequisites

Node.js installed (for npx), OR a VS Code extension like Live Server.

Installation

Clone the repository:

git clone [https://github.com/Rachit-Mohanty/expflow.git](https://github.com/Rachit-Mohanty/expflow.git)
cd expflow


Run a local server:
If you have Node.js, you can run:

npx serve .


Alternatively, open index.html using the "Live Server" extension in VS Code.

Open your browser:
Navigate to http://localhost:3000 (or the port provided by your server).
